# Network IDS

Лёгкая IDS на Python для малых сетей: анализирует трафик в реальном времени, обнаруживает аномалии и блокирует подозрительные IP-адреса.

## Возможности

- Обнаружение аномалий по входящему трафику;
- Защита от:
  - **SYN-Scan**,
  - **TCP Flood**,
  - **UDP Flood**,
  - **ICMP Flood**;
- Адаптивный порог с автоматической подстройкой под трафик;
- Дополнительная проверка IP по репутации перед блокировкой;
- Логирование всех срабатываний.

## Кратко о принципе работы

1. Система перехватывает входящие пакеты.
2. Считает количество пакетов в скользящем окне.
3. Динамически обновляет порог обнаружения.
4. При многократном превышении порога проверяет IP на репутацию.
5. Записывает событие в лог и блокирует адрес.

## Требования

- Python **3.13**
- Linux
- права на выполнение сетевого мониторинга и управления firewall

## Установка (Linux)

Для установки программы введите следующие команды в терминал.
```bash
sudo apt update
sudo apt install curl

curl -sSL https://raw.githubusercontent.com/vsosh44/ids/refs/heads/main/install.sh | bash
```

## Логи

Программа сохраняет все аномалии в сети в файл `/var/log/network_ids/network_ids.log`

Прочитать его можно, используя следующую команду

```bash
less /var/log/network_ids/network_ids.log
```
